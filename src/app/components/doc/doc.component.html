API

Genres

GET

ดึง Genres ทั้งหมด

http://sittikiat.streetfood.in.th:1111/api/genres
Response
  • genre_id รหัสหมวดหมู่
  • genre_name ชื่อหมวดหมู่
  • genre_pic รูปภาพหมวดหมู่


Albums

GET

ดึง Albums ทั้งหมด

http://sittikiat.streetfood.in.th:1111/api/albums
Response
  • album_id รหัสอัลบั้ม
  • album_name ชื่ออัลบั้ม
  • album_pic รูปภาพอัลบั้ม

GET

ดึงเฉพาะบาง Album

http://sittikiat.streetfood.in.th:1111/api/album/:id

ต้องการ id ของ album

ตัวอย่าง

http://sittikiat.streetfood.in.th:1111/api/album/1

Response
  • song_id รหัสเพลง
  • song_name ชื่อเพลง
  • song_file ไฟล์เพลง
  • song_pic รูปภาพเพลง
  • artist_id รหัสศิลปิน
  • album_id รหัสอัลบั้ม
  • album_name ชื่ออัลบั้ม
  • album_pic รูปภาพอัลบั้ม
  • genre_id รหัสหมวดหมู่
